Instructions

Each card as 4 features:

  • Type: cool, laugh, hug
  • Color: yellow, blue, red
  • Number: 1, 2, 3
  • Shadow: left, right, none

A 'Set' consists of three cards in which each feature is EITHER the same on each card OR is different on each card. That is to say, any feature in the 'Set' of three cards is either common to all three cards or is different on each card.

Example of a good set

All three cards have a cool emoji, all have different colors, all have two emojis, and all have different shadows.

Example of a wrong set

All three cards have same color, all have different number of emojis, and all have same shadow.
BUT there is 2 hug emojis and a cool one, it is neither all the same nor all different.
